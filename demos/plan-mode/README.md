# Plan Mode Demo

## 场景
规划一个 Bash 脚本工具。

## 步骤

1. 输入: "规划一个简单的 Bash 脚本工具"
2. Copilot 输出规划步骤:
   - 定义脚本功能
   - 添加参数解析
   - 实现核心逻辑
   - 添加帮助信息

## 使用方法
- **激活方式**: 集成在聊天中
- **技巧**: 用于项目规划和步骤分解
- **与 Agent Mode 的不同**: Plan Mode 专注于规划和策略，而 Agent Mode 专注于执行

## 规划输出示例

```
# Bash 脚本工具规划

## 1. 功能定义
- 读取输入文件
- 处理数据
- 生成输出

## 2. 参数解析
- 支持 --input 参数
- 支持 --output 参数
- 支持 --help 选项

## 3. 核心逻辑
- 验证输入文件
- 执行数据处理
- 输出结果

## 4. 帮助信息
- 显示使用说明
- 列出所有选项
```

## 优势
- 清晰的项目结构
- 系统化的开发流程
- 完整的功能规划
- 提高开发效率
