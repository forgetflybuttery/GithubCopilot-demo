# Agent Mode Demo

## 场景
运行 Bash 脚本测试。

## 步骤

1. 在聊天中输入: `/run bash test.sh`
2. Copilot 执行脚本并返回结果。

## 使用方法
- **激活方式**: 通过聊天面板或特定命令激活
- **技巧**: 用于自动化工作流，如代码审查或测试生成
- **演变**: 从被动建议演变为主动任务执行

## 特性
- 直接执行命令
- 实时获取输出结果
- 自动化工作流
- 集成测试和验证

## 命令格式
```
/run bash <script-name>
/run python <script-name>
/run <other-commands>
```

## 优势
- 实时反馈
- 快速迭代
- 自动化部署
- 错误检测和修复
